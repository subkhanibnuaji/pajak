# PAJAKKU v2.0

**Knowledge Base Perpajakan Indonesia yang Komprehensif**

Personal knowledge base ("second brain") perpajakan Indonesia yang mencakup seluruh jenis pajak dengan regulasi terkini, kalkulator interaktif, dan studi kasus.

## Tech Stack

- **Framework**: Next.js 15 (App Router, TypeScript strict)
- **Styling**: Tailwind CSS v4 + shadcn/ui
- **State**: Zustand + localStorage + IndexedDB
- **Search**: Command palette (cmdk)
- **Deployment**: Vercel (free tier, zero cost)
- **Content**: File-based (TypeScript data files)

## Features

- Knowledge base lengkap: PPh, PPN, KUP, PBB, BPHTB, Pajak Daerah, Transfer Pricing, Pajak Digital
- Kalkulator interaktif: PPh 21 (TER & Pasal 17), PPN, PPh Badan, BPHTB
- Dark mode, responsive, accessible
- Command palette search (Ctrl+K)
- Regulasi terkini: UU HPP (7/2021), PP 58/2023, PMK-168/2023, Coretax 2025

## Development

```bash
npm install
npm run dev
```
